Reinforcement Learning Based Floating Object Grasping Using a 6-DOF Robotic Arm

Khiem Nguyen, Mooi Choo Chuah,
Lehigh University
Code

Our reinforcement learning based floating object grasping method effectively adapts to the unpredictable object movements with different velocities and directions in a 3D space.

Abstract

We present a novel application of reinforcement learning to the complex task of grasping a floating object with random movements in a 3D space using a robotic arm with 6 degrees of freedom (6-DOF).

Traditional grasping methods cannot perform well in locating unpredictable positions of a floating object in an underwater environment. To address the challenge of dynamic and unpredictable object behaviors, our proposed technique leverages an improved Deep Deterministic Policy Gradients (DDPG) deep reinforcement learning (DRL) algorithm to enable the robotic arm to learn effective grasping strategies through iterative interactions, all without the need for velocity control. Our framework decouples object detection from DRL to facilitate the transfer of a simulation environment to a real robot. Our experimental results with the Unity simulator incorporating real-time motion control using the MoveIt Motion Planning Framework demonstrate the efficacy of our approach over others, such as the point-to-point approach, without using reinforcement learning.
